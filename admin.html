VR Hotspots Admin

Use these tools to manage classes, shared media, student project submissions, and usage limits. Sign in with your admin password on any admin page. Students use the main editor at /; this area is for teachers and site administrators only.

Choose a section below or use the navigation bar at the top of any admin page.

Submissions

Review projects students submit for grading. Download ZIPs, host tours online, return work with feedback, import backups, and sync with cloud storage.

Review All Content

Site-wide view of student projects, uploads, shared online assets, templates, and orphaned files — with unified delete and storage cleanup.

Assets

Manage the shared asset library (360° images, videos, audio, flat images, models). Upload files students can pick from the editor’s Asset Library.

Users

Create classes and student accounts, reset passwords, export login lists, and peek into a student’s submissions and personal assets.

Billing & Limits

View usage per class and set limits for submissions, storage, hosted projects, and asset uploads. Override tier defaults or set limits to unlimited without Stripe.

Quick tips

  • Set up Users first — classes, students, and passwords — before students sign in.
  • Upload shared media in Assets so students can reuse approved files.
  • Students submit from the editor; new work appears in Submissions.
  • If submissions are blocked, check Billing & Limits for monthly caps.
